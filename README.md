# E-Commerce Product Return Analytics & Customer Satisfaction-PowerBI

## Project Overview
This project analyzes product return patterns in an e-commerce business to identify the key factors influencing returns and their impact on customer satisfaction and profitability.

## Problem Statement
High product return rates lead to revenue loss and reduced customer satisfaction. The objective of this project is to identify the major drivers of returns and provide actionable business recommendations.

## Tools Used
- Microsoft Excel (Data Cleaning & Preprocessing)
- Power BI (Dashboard Development & Visualization)

## Dataset
- 100,000+ records
- 17+ columns
- Features include Product Category, Region, Discount %, Delivery Days, Customer Rating, Return Status, and more.

## Dashboard Pages

### Dashboard 1: Return Overview
- Total Revenue
- Total Orders
- Return Rate
- Return Loss
- Category-wise and Region-wise Analysis

### Dashboard 2: Return Behavior & Root Cause Analysis
- Customer Rating vs Returns
- Delivery Time vs Returns
- Discount Impact Analysis
- Order Quantity Trends

### Dashboard 3: Return Impact & Profit Improvement
- Revenue vs Return Loss
- Category-wise Return Loss
- Region-wise Analysis
- Monthly Return Trends

## Key Insights
- Fashion products showed the highest return rates.
- Delivery delays significantly increased return probability.
- Higher discounts were associated with more returns.
- Lower-rated products experienced higher return rates.
- Return patterns varied across regions.

## Business Recommendations
- Improve product quality in high-return categories.
- Optimize delivery performance.
- Review discount strategies.
- Improve customer satisfaction and ratings.
- Strengthen logistics operations in high-return regions.
